# 开始使用

欢迎来到这套博客模板。

它现在是一份 **博客 + 文档混合结构**，适合以后按下面几种方式继续扩展：

1. 把 `notes/` 当成日常维护和折腾记录
2. 把 `guide/` 当成长期保留的整理文档
3. 后面再逐步补专题页、归档页、友情链接页

## 当前结构

```text
ElicsdyBlog/
├─ docs/
│  ├─ README.md
│  ├─ guide/
│  ├─ notes/
│  └─ .vuepress/
└─ .github/workflows/
```

## 你现在能直接改的地方

- 首页内容：`docs/README.md`
- 导航与侧边栏：`docs/.vuepress/config.js`
- 关于页：`docs/about/README.md`
- 笔记与文章：`docs/notes/`、`docs/guide/`
